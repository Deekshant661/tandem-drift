feat: Phase 3 - Fernvale, a handcrafted showcase drive with real Kenney assets

New showcase map (~758m, verified self-intersection-free), designed
road-first per the Phase 3 spec: village -> farmland -> tight 90-degree
corner with flower beds -> narrow bridge approach -> red covered bridge ->
blind-crest into forest -> forest hairpin -> lake reveal (dock + lookout)
-> fast sweeping lakeshore corner -> windmill fields -> chicane -> long
sunset-facing straight -> back to village. Road width varies continuously
by place (5.5 half-width at the tight corner, 4 at the bridge, 10 in the
village) using the same generic spline/wall pipeline Phase 2 built --
zero server or physics changes.

Real assets: 44 CC0 glTF models (1.7 MB total) sourced from six Kenney
kits (Nature, Fantasy Town, City Suburban, Car, Furniture, Pirate),
covering trees, rocks, fences, a bridge, a windmill, a dock, parked
vehicles, and houses. The instancing pipeline built in Phase 2 for
procedural scatter is extended to resolve templates from async-loaded
glTF as well, so real and procedural pieces (mailbox, lamp, hay bale
reused from Willowbrook) share one GPU-instanced path with no
special-casing.

Continuous atmosphere: fog color/distance, ambient tint, vegetation
density, and palette bias all blend smoothly across named character
waypoints (village/farmland/forest/lakeside/windmill fields) by
arc-length position -- no hard zone switches, verified by a
continuity test. Ambient audio mirrors the same blend with four soft
area loops crossfaded by the car's position on the road.

Terrain: a single-purpose heightfield is exactly 0 within the road
corridor (+ margin) and blends into hand-authored hills/valley/cliff
well beyond it, so the road never visually climbs while the
surroundings gain real depth -- caught and fixed a genuine sign-error
bug in the cliff-blend direction via its own test before it ever
reached the world.

Composition follows the "negative space over density" rule throughout:
~160 total placements (landmarks + repeated scenery) vs. Willowbrook's
600+, with an explicit rejection-sampling guard so clusters never land
on the drivable road. Named orientation landmarks: an oversized
windmill, the red covered bridge, a lakeside dock + canoe, a hilltop
lookout, a hero oak, and a distant composed church/tower silhouette.
Cheap ambient life throughout: GPU-shader wind sway on all foliage (one
time-uniform write per frame, zero per-instance CPU cost), a simple
animated lake shimmer, a handful of birds/butterflies/fireflies/chimney
smoke.

Registered as a new selectable map (fernvale) alongside Willowbrook,
which is untouched and stays the regression/test map and server
default. Vehicle render scale bumped 12% (visual only, physics
untouched) per the explicit toy-like-car preference.

40 new tests (heightfield, Fernvale road validity + self-intersection +
perimeter, atmosphere continuity, hand-instancing, scenery placement),
128 total passing. Verified end-to-end over real WebSockets (two-client
pilot+engineer drive test) against a local server.
